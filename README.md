## Como executar
- Navegue ate a pasta ServidorCentral pelo terminal
- Execute no terminal make
- Execute o código com ./prog
- Agora em uma nova aba do terminal navegue até o ServidorDistribuido
- Execute no terminal make
- Execute o código com ./prog
- Os 2 servidores devem estar ativos
